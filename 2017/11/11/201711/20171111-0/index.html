lola 와 lol 은 또 뭐람

Hexo
2017-11-11
git

시작

그냥 평소처럼 생활코딩에 글을 올렸는데, 답글 중에서 이런 답글이 달렸다.

분명 아무것도 안 건드린 git 은 이런 명령어를 지원하지 않을 터.. 대체 뭔가 하고 검색을 해보기로 했다. 답글 달리기 까지는 시간이 엄청 걸릴게 뻔하니까

전개

구글 선생님의 검색기를 빌려서 검색해본 결과 리눅스에서 지원하는 .gitconfig 의 내용들이 눈에 띄었다.

추측하기로는 lola 와 lol 은 git 의 aliasing 기능을 이용해서 일종의 숏컷을 만든 것 같았다. git 을 console 에서 자주 쓰는 나에게 꽤 좋은 기능인 것 같아 Windows 환경에서 숏컷을 만들어보기로 했다.

그 후에 git 레포지토리가 있는 디렉토리에 들어가 시험해본 결과 잘 작동됬다.
